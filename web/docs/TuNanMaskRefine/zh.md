# 图南蒙版微调

> 如果你是从 ComfyUI Manager 安装到这里，请记得同时安装 Photoshop 插件。  
> 下载入口：
> - 官网：[tunanart.cn](https://tunanart.cn)
> - GitHub Releases：[tunan-paint-bridge Releases](https://github.com/beiyue0208ly/tunan-paint-bridge/releases)

这是给选区和蒙版做快速后处理的工具节点。

## 作用

- 扩张或收缩蒙版边缘
- 给蒙版做柔化
- 支持反转蒙版

## 常见用途

- 让局部重绘边缘更自然
- 修正 Photoshop 传入选区的边缘
- 在回贴或重绘前做一层轻量蒙版整理
